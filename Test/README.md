# How to Start Server:

If you don't have the dependencies already, firstly do the following:

1. `npm i`
2. `cd client`
3. `npm i`

## Starting / restarting the server

1. `cd client`
2. `npm run build`
3. `cd ..`
4. `npm run dev`
5. Express server located on `http://localhost:5000/`
6. React client located on `http://localhost:3000/`
